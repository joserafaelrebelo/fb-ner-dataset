Descricão
###
Hoje temos mães mais informadas que profissionais.  Se uma mãe chegasse para você hoje dizendo:  Minha filha(o) tem plagiocefalia occipital direita!  Você realmente entenderia?  Para contextualizar melhor, a plagiocefalia é quando uma das partes do crânio fica amassada, onde um lado parece mais profundo que o outro.  Não é necessário entender o grau exato caso você não seja fisioterapeuta, mas saber para quem encaminhar é de extrema importância.   Isso muitas vezes está ligado à anquiloglossia, gerando diversas compensações para que o bebê “consiga” mamar, utilizando não só a boca, mas o corpo inteiro.  Essa é a importância de saber avaliar, como é o tratamento e para quem encaminhar.  Salva para lembrar sempre
